# qw-admin

# 一、介绍

基于 Spring Boot、Shiro 的快速开发管理系统。

> 现阶段未单体项目，后期考虑采用 java9 之后的模块化拆分系统。

# 二、软件架构

- 后端技术
  - Spring Boot 2.0.0
  - Apache Shiro
  - Spring Data JPA
  - Mybatis Plus
  - Thymeleaf
  - Ehcache
  - ...
- 前端技术
  - requirejs
  - vue.js(2+)
  - vue-router
  - vue-store
  - jquery
  - antd-vue
  - antd-vue-pro
  - lodash
